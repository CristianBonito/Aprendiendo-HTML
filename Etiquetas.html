¿Qué es una etiqueta en HTML?

Una etiqueta en HTML es un elemento que se utiliza para estructurar y organizar el contenido de una página web. Las etiquetas son instrucciones que le indican al navegador cómo debe mostrar el contenido al usuario. Cada etiqueta generalmente tiene un nombre, y se utiliza en pares: una etiqueta de apertura y una etiqueta de cierre.

¿Para qué sirven las etiquetas en HTML?

Las etiquetas en HTML son esenciales para definir la estructura y el formato del contenido en una página web. Permiten organizar texto, imágenes, enlaces, formularios y otros elementos, facilitando la navegación y la accesibilidad. Además, el uso correcto de etiquetas semánticas mejora el SEO (optimización para motores de búsqueda) y la comprensión del contenido por parte de los lectores de pantalla.

Etiqueta <h1> a <h6>

Estas etiquetas se utilizan para definir encabezados de diferentes niveles en el documento HTML.

El uso de encabezados ayuda a estructurar el contenido y mejora la accesibilidad, ya que los lectores de pantalla utilizan estas etiquetas para navegar por el contenido.

Es importante usar estas etiquetas de manera jerárquica, comenzando con <h1> como el encabezado principal y utilizando <h2>, <h3>, etc., para subtítulos y secciones subsecuentes.

Uso

<h1> Encabezado principal </h1>
    <h2> Encabezado secundario </h2>
    <h3> Encabezado terciario </h3>
    <h4> Encabezado cuaternario </h4>
    <h5> Encabezado quinario </h5>
    <h6> Encabezado senario </h6>

Etiqueta <p>

Esta etiqueta se utiliza para definir un párrafo en el texto. Los párrafos ayudan a agrupar oraciones relacionadas y a facilitar la lectura.

Los navegadores suelen agregar un margen alrededor de los párrafos, lo que mejora la legibilidad.

Es recomendable no utilizar párrafos vacíos, ya que pueden afectar la presentación del contenido en algunos navegadores.

Uso

<p> Este es un párrafo </p>

Etiqueta <strong>

Esta etiqueta se utiliza para resaltar texto en negrita, indicando que el contenido tiene un significado importante. También ayuda en la accesibilidad al enfatizar contenido relevante.

El uso de <strong> puede también afectar el estilo visual, pero su objetivo principal es semántico.

Uso

<strong> Texto en negrita </strong>

Etiqueta <em>

Esta etiqueta se utiliza para enfatizar texto, generalmente mostrado en cursiva. Su propósito es destacar ciertas palabras o frases dentro de un contexto.

Al igual que <strong>, <em> también tiene un significado semántico, indicando que el contenido es relevante en el contexto.

Uso

<em> Texto en cursiva </em>

Etiqueta <a>

Esta etiqueta se utiliza para definir un enlace a otra página o recurso. Los enlaces son fundamentales para la navegación en la web.

El atributo target se puede usar para especificar cómo se abrirá el enlace, por ejemplo, en una nueva pestaña.

Uso

<a href="URL"> Enlace </a>

El atributo href especifica la dirección a la que se dirige el enlace.


Etiqueta <ul>

Se utiliza para crear una lista desordenada, donde los elementos se presentan con viñetas. Esta etiqueta es útil para agrupar elementos que no requieren un orden específico.

Las listas pueden anidarse dentro de otras listas, creando jerarquías de información.

Uso

<ul>
    <li> Elemento de lista 1 </li>
    <li> Elemento de lista 2 </li>
    </ul>

Etiqueta <ol>

Se utiliza para crear una lista ordenada, donde los elementos se presentan con números. Es ideal para mostrar secuencias o pasos en un proceso.

Las listas ordenadas también pueden anidarse para crear sub-listas.

Uso

<ol>
    <li> Elemento de lista numerada 1 </li>
    <li> Elemento de lista numerada 2 </li>
    </ol>

Etiqueta <li>

Se utiliza para definir un elemento dentro de una lista, ya sea desordenada u ordenada. Los elementos de lista mejoran la claridad y la organización de la información.

Los elementos de lista deben estar contenidos dentro de etiquetas <ul> o <ol>.

Uso

<li> Elemento de lista </li>

Etiqueta <img>

Esta etiqueta se utiliza para insertar una imagen en la página. La inclusión de imágenes en el contenido web puede enriquecer la experiencia del usuario.

Es importante usar el atributo alt para describir la imagen, lo cual es crucial para la accesibilidad y SEO.

Uso

<img src="URL" alt="Descripción">

El atributo alt proporciona una descripción de la imagen, lo cual es importante para la accesibilidad y en caso de que la imagen no se cargue.


Etiqueta <video>

Se utiliza para insertar un video en la página. Esta etiqueta permite una variedad de atributos que controlan la reproducción y visualización del video.

Se pueden incluir múltiples fuentes de video para asegurar la compatibilidad con diferentes navegadores.

Uso

<video src="video.mp4" controls></video>

El atributo controls permite mostrar los controles de reproducción, como play, pause y volumen.


Etiqueta <audio>

Se utiliza para insertar un archivo de audio en la página. Similar a la etiqueta de video, permite a los usuarios reproducir sonido directamente en su navegador.

Es posible incluir múltiples formatos de audio para mejorar la compatibilidad entre navegadores.

Uso

<audio src="audio.mp3" controls></audio>

El atributo controls muestra controles para la reproducción del audio.


Etiqueta <form>

Se utiliza para crear un formulario que permite a los usuarios enviar datos. Los formularios son esenciales para la interacción del usuario en aplicaciones web.

El uso de etiquetas <label> junto con <input> mejora la accesibilidad y la usabilidad del formulario.

Uso

<form action="URL" method="POST">
    <input type="text" placeholder="Nombre">
    <textarea> Escribe aquí... </textarea>
    <button> Enviar </button>
    </form>

Etiqueta <input>

Se utiliza para crear campos de entrada en un formulario. Esta etiqueta es versátil y permite varios tipos de entrada, como texto, contraseñas, correos electrónicos, etc.

Es esencial utilizar el atributo type para definir el tipo de entrada esperada.

Uso

<input type="text" placeholder="Ingrese texto">

Etiqueta <button>

Se utiliza para crear un botón en el formulario. Los botones pueden ser utilizados para enviar el formulario o para ejecutar acciones específicas mediante JavaScript.

El atributo type se puede establecer como submit, reset o button, dependiendo de la funcionalidad deseada.

Uso

<button type="submit"> Enviar </button>

Etiqueta <select>

Se utiliza para crear un menú desplegable en un formulario. Permite a los usuarios seleccionar una opción de una lista predefinida.

El uso de etiquetas <option> permite definir las opciones disponibles en el menú.

Uso

<select>
    <option value="1"> Opción 1 </option>
    <option value="2"> Opción 2 </option>
    </select>

Etiqueta <input type="checkbox">

Se utiliza para crear una casilla de verificación en el formulario. Los usuarios pueden seleccionar una o más opciones.

El uso de casillas de verificación es común en formularios donde se requiere elegir múltiples opciones.

Uso

<input type="checkbox" name="opcion1"> Opción 1

Etiqueta <input type="radio">

Se utiliza para crear un botón de opción en el formulario. Los botones de opción permiten seleccionar una sola opción de un conjunto de opciones.

Es importante agrupar los botones de opción utilizando el mismo atributo name para asegurar que solo una opción esté seleccionada a la vez.

Uso

<input type="radio" name="grupo1"> Opción 1

Etiqueta <textarea>

Se utiliza para crear un campo de texto de varias líneas en un formulario. Permite a los usuarios ingresar texto extenso.

El uso de esta etiqueta es útil cuando se requiere más espacio para la entrada de texto, como en comentarios o descripciones.

Uso

<textarea placeholder="Escribe aquí..."></textarea>
