CSS: Hojas de Estilo en Cascada

¿Qué es CSS?

Cascading Style Sheets (CSS) es un lenguaje de diseño utilizado para describir la presentación de un documento escrito en HTML o XML. CSS permite a los desarrolladores y diseñadores web controlar el diseño, el formato y la apariencia de los elementos en una página web de manera eficiente.

¿Para qué sirve CSS?

CSS se utiliza para:

  • Separar el contenido de la presentación, facilitando la mantenibilidad del código.
  • Definir estilos para diferentes elementos HTML, como colores, fuentes, márgenes y posiciones.
  • Crear diseños responsivos que se adaptan a diferentes dispositivos y tamaños de pantalla.
  • Aplicar efectos visuales como sombras, transiciones y animaciones.

¿Cómo se usa CSS?

CSS se puede utilizar de tres maneras:

  1. CSS en línea

    Se aplica directamente en el elemento HTML utilizando el atributo style.

    <p style="color: blue; font-size: 20px;">Este es un párrafo azul.</p>

  2. CSS interno

    Se incluye en la sección <head> del documento HTML usando una etiqueta <style>.

    <style>
    p {
        color: green;
        font-size: 18px;
    }
</style>

  3. CSS externo

    Se guarda en un archivo separado con extensión .css y se vincula al documento HTML usando una etiqueta <link>.

    <link rel="stylesheet" href="styles.css">

Ejemplos de CSS

Establecer colores y fuentes

body {
    background-color: #f0f0f0;
}

h1 {
    color: #333;
    font-family: Arial, sans-serif;
}

Este es un encabezado

Este párrafo tiene un fondo claro y texto oscuro.

Crear un diseño responsivo

@media screen and (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

Este texto cambia de tamaño en dispositivos pequeños

Reduce el tamaño de la fuente cuando el ancho de la pantalla es menor a 600px.

Características avanzadas de CSS

  • Flexbox: Un modelo de diseño que permite crear estructuras de página flexibles y adaptativas.
  • Grid: Un sistema de cuadrícula que facilita la creación de diseños complejos de manera intuitiva.
  • Animaciones y Transiciones: Permiten aplicar efectos de movimiento a los elementos cuando cambian de estado.
