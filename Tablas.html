Tablas en HTML: Creación y Estilos

Las tablas en HTML se utilizan para organizar y mostrar información de manera estructurada en filas y columnas. Aunque anteriormente se usaban para diseñar el layout de sitios web, esa práctica ha sido reemplazada por el uso de CSS. Hoy en día, las tablas son esenciales para mostrar datos que requieren estructura y legibilidad, como estadísticas, calendarios y reportes financieros.

Estructura básica de una tabla

Para crear una tabla en HTML, necesitas entender las etiquetas clave que definen su estructura:

  • <table>: Esta etiqueta envuelve el contenido de la tabla y define el inicio y fin de la misma.
  • <tr>: Representa una fila en la tabla. Cada fila contiene celdas con datos o encabezados.
  • <th>: Define una celda de encabezado de tabla. Usualmente se utiliza en la primera fila o columna para describir el contenido de las celdas bajo o a la derecha de esta.
  • <td>: Define una celda de datos. Es el contenedor de cada unidad de información dentro de la tabla.

Ejemplo básico de tabla

<table>
<tr>
  <th>Nombre</th>
  <th>Edad</th>
  <th>Ciudad</th>
</tr>
<tr>
  <td>Juan</td>
  <td>25</td>
  <td>Ciudad de México</td>
</tr>
<tr>
  <td>María</td>
  <td>30</td>
  <td>Guadalajara</td>
</tr>
</table>

Salida:

Nombre Edad Ciudad
Juan 25 Ciudad de México
María 30 Guadalajara

Atributos comunes en las tablas

Las tablas tienen varios atributos que permiten personalizar su apariencia y funcionalidad.


1. border (Borde)

El atributo border define el grosor del borde de la tabla. Aunque este atributo se usaba en versiones más antiguas de HTML, actualmente es mejor manejar los bordes usando CSS para un mayor control y personalización.

<table border="1">
    <tr>
      <th>Producto</th>
      <th>Precio</th>
    </tr>
    <tr>
      <td>Camisa</td>
      <td>$20</td>
    </tr>
    </table>

Salida:

Producto Precio
Camisa $20

2. colspan y rowspan

Estos atributos permiten que una celda se extienda a lo largo de varias columnas o filas, respectivamente, lo que es útil para agrupar datos relacionados.

  • colspan: Extiende una celda en varias columnas.
  • rowspan: Extiende una celda en varias filas.
Ejemplo de colspan:
<table border="1">
    <tr>
      <th colspan="2">Encabezado Combinado</th>
    </tr>
      <tr>
      <td>Producto</td>
      <td>Precio</td>
    </tr>
    </table>

Salida:

Encabezado Combinado
Producto Precio

Ejemplo de rowspan:
<table border="1">
    <tr>
      <th rowspan="2">Categoría</th>
      <td>Frutas</td>
    </tr>
    <tr>
      <td>Verduras</td>
    </tr>
    </table>

Salida:

Categoría Frutas
Verduras

Estilizando tablas con CSS

Una de las ventajas del uso moderno de HTML y CSS es la capacidad de darle estilo a las tablas, mejorando su presentación y adaptándola a las necesidades visuales del sitio.


1. Ancho y alto de la tabla

Para ajustar el tamaño de la tabla, puedes usar las propiedades width y height en CSS.

<table style="width: 100%; height: 200px;">
    <tr>
      <th>Producto</th>
      <th>Precio</th>
    </tr>
    <tr>
      <td>Camisa</td>
        <td>$20</td>
          </tr>
        </table>

Salida:

Producto Precio
Camisa $20

2. Ajuste de espaciado entre celdas

Con border-spacing, puedes ajustar el espacio entre las celdas de la tabla.

<table style="border-spacing: 10px;">
    <tr>
      <th>Producto</th>
      <th>Precio</th>
    </tr>
    <tr>
      <td>Camisa</td>
      <td>$20</td>
    </tr>
  </table>

Salida:

Producto Precio
Camisa $20

3. Colapsar bordes

Con el estilo border-collapse, puedes colapsar los bordes de la tabla.

<table style="border-collapse: collapse;" border="1">
    <tr>
      <th>Producto</th>
      <th>Precio</th>
    </tr>
    <tr>
      <td>Camisa</td>
      <td>$20</td>
    </tr>
    </table>

Salida:

Producto Precio
Camisa $20

4. Colores y fondo

Usando background-color y color, puedes personalizar los colores de fondo y del texto.

<table border="1">
    <tr style="background-color: #00425a; color: white;">
      <th>Producto</th>
      <th>Precio</th>
    </tr>
    <tr>
      <td>Camisa</td>
      <td>$20</td>
    </tr>
    </table>

Salida:

Producto Precio
Camisa $20

5. Alineación de contenido

Puedes usar text-align para alinear horizontalmente el texto y vertical-align para alinear el contenido dentro de las celdas.

<table border="1">
    <tr>
      <th style="text-align: left;">Producto</th>
      <th style="text-align: right;">Precio</th>
    </tr>
    <tr>
      <td style="vertical-align: middle;">Camisa</td>
      <td>$20</td>
      </tr>
    </table>

Salida:

Producto Precio
Camisa $20

Las tablas en HTML son una herramienta poderosa para organizar datos. Aunque su diseño ha evolucionado con el tiempo, siguen siendo fundamentales para presentar información tabular. Con CSS, es posible personalizar su apariencia, mejorando tanto la estética como la funcionalidad, y con buenas prácticas de accesibilidad, puedes asegurarte de que toda tu audiencia pueda interactuar con ellas correctamente.
